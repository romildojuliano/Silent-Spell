## Instalação
No terminal, digite:

`git clone https://github.com/romildojuliano/Silent-Spell.git`

`cd Silent-Spell`

Na mesma pasta, instale os dependências com o comando `npm install`

Para inicializar o projeto, digite `expo starT`
